Capabilities

Engineered for real work.

Thermal Coupling

BJT, MOSFET, diode, and inductor temperatures feed back into the electrical model. Watch runaway, Curie-point saturation, and core losses in real time.

AC-MNA Analysis

Bode and Nyquist plots computed via complex MNA. One linear solve per frequency — 155× faster than transient sweeps.

Professional Multimeter

Drag probes onto pins or wires. Auto AC/DC detection. V, I, R, dB modes. MIN, MAX, and PEAK hold.

Under the hood

Built on fundamentals.

No shortcuts, no approximations that don't hold up under probing. Physics, faithfully.

Solver

  • Modified Nodal Analysis (MNA)
  • Trapezoidal and Backward Euler integration
  • Adaptive Newton-Raphson with divergence rollback
  • Complex MNA for AC small-signal
  • Junction temperature coupling across all nonlinear devices
  • Sparse LU factorisation with dense fallback

Device Models

  • Gummel-Poon BJT with bandgap Is(T)
  • BSIM3 and SPICE Level-1 MOSFET
  • Shockley diode with configurable Eg
  • Inductor saturation, hysteresis, and eddy-current loss
  • Curie-point derating of magnetic cores
  • Behavioral sources, SCRs, transformers, transmission lines
v12.0.0-alpha.1 · April 2026

The Great Reset.

All 55 bundled preset circuits removed. Preset-based testing abandoned. The foundation will be rebuilt feature-by-feature — each component, source, model, and analysis verified against analytic or LTspice reference before any preset returns.

🪦

55 presets archived

PRESETS is now the empty array. Four preset- dependent test files removed: preset-roundtrip, preset-integrity, preset-anchors, preset-geometry. The gallery panel is empty until presets re-earn their place.

🧭

Feature-first test discipline

New src/test-spice/feature-tests/ directory. One folder per component / source / model / analysis. Each carries an LTspice-verified netlist, a puppeteer probe that loads it via SpiceImport (never loadPreset), and a short expected.md.

⚖️

Five fences → three

npm run test:presets and npm run test:integrity removed. Remaining CI surface is tight and honest: harness 11/11, scenarios (13 files), and test:sparse 25/25. No green checks for things that weren't really tested.

🧱

Four-iteration lesson, settled

Sprints 98, 101, 102, 103 each caught a different way tests can lie: race conditions, missing coverage, shared-bug round-trips, false-convergence. This release accepts the lesson by rebuilding from the fence up.
